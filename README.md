
### How to install and run
```
git clone https://github.com/Scott-McKool/CEN4020_Project2.git

cd CEN4020_Project2/

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python app.py
```

then the terminal will tell you where the app is hosted (like 127.0.0.1:5000). visit the address in a web browser.
